# SayeOS - Open Source Distribution

**Constitutionally Governed AI Agents**

The open-source home of SayeOS - a new operating system for AI agents grounded in constitutional principles.

## Why Constitutional Governance?

AI is moving from tools to autonomous agents. Without clear separation of powers, we risk the same problems that plagued early computing and the industrial revolution: unchecked power, opacity, and unintended harm.

The U.S. Constitution solved this for human governments 237 years ago. SayeOS applies the same proven framework to AI:
- Separation of concerns (Intent → Thought → Action)
- Built-in checks, balances, and human veto rights
- Governance that is foundational, not bolted on later

## The Triad Architecture

Constitutional governance through separation of concerns:

### Intent
**You decide what you want.** Set goals, define boundaries, establish priorities. This is the human layer - where direction comes from.

### Thought
**AI reasons through options.** Research, planning, trade-off analysis. All reasoning stays visible and auditable. You can question, redirect, or reject.

### Action
**Only approved steps execute.** Nothing happens automatically. You review and approve before any action is taken in the real world.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/sayeos/sayeos-os.git

# Install dependencies
cd sayeos-os
./install.sh

# Run the CLI
sayeos --help
```
